## Strategy / Research Interpretation

1. **The current 20-bar minute momentum family remains structurally weak under this evaluator.** Many filters reduced or reshaped exposure, but the score distribution stayed deeply negative: max `-4.7001`, median `-46.6556`.
2. **Small filters on top of the same stack are likely exhausted for now.** The run repeatedly tested no-trade bands, volatility normalization, trend alignment, VWAP fade, universe stability, and participation gates without a keep.
3. **Universe selection alone did not overcome signal weakness.** Liquidity, ETF exclusion, daily range/stability tilts, and related daily-context screens all reverted.
4. **The best candidate being negative but less bad is still useful evidence.** Iteration `11` should be mined as a cautionary near-miss, not promoted: its score `-4.7001` remained far below positive baseline comparison.

## Recommended Next Research Direction

Do **not** keep iterating small variants of the same sign-only 20-bar momentum stack as the next default path. Recommended next round:

1. **Switch the primitive, not just the filter.** Test a genuinely different intraday signal primitive, for example opening-range breakout / intraday reversal by session segment / volatility-breakout with explicit time-of-day constraints.
2. **Add explicit transaction-cost / churn diagnostics to the report loop.** Many failures likely share churn/overtrading symptoms; each iteration report should surface trades, win rate, profit factor, drawdown, and per-symbol concentration, not only SCORE.
3. **Cluster failed idea families before launching another long run.** The continuation memory now has many similar rejected branches. Add a pre-run anti-repeat gate that blocks semantically equivalent momentum-filter variants unless the primitive changes.
4. **Keep OMX as the engine.** The runtime itself completed the five-hour validation; the research failure is strategy-quality, not orchestration failure.
